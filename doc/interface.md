# 意图识别

## 请求方法

post

## 请求格式

json

## 请求地址

[http://172.28.21.195:6400/intent](http://172.28.21.195:6400/intent)

## 请求参数

|字段|说明|类型|备注|是否必填|
|:----|:----|:----|:----|:----|
|query|查询内容|string|    |是|

## 返回参数

|字段|说明|类型|备注|是否必填|
|:----|:----|:----|:----|:----|
|label|返回标签|string|若为chitchat，返回固定模版<br>若为其他调用问答、实体链接接口|是|
|score|概率评分|float|    |是|
